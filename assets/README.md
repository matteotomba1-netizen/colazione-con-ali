# Brand Assets

No proprietary logo files or photography were provided. The brand mark is typographic:

- **Mark:** `✦` (U+2726) set in a circle — black, or filled with `--grad-warm` for accent placements.
- **Wordmark:** "Colazione con Ali" in Inter, wght 500, -0.4px tracking.
- **Logo behaviour:** the mark rotates 90° + scales 1.15 on hover (`--ease-spring`).

Recipe/run imagery is real photography (Reel thumbnails) when available; otherwise use `RecipeCard`'s sunrise **gradient fills** (warm/peach/cool/sky/berry/cream) as placeholders.

Iconography is inline stroke SVG + Unicode marks (`✦ ♡ → ↗ ·`) + food emoji. See README → ICONOGRAPHY. Drop any future SVG/PNG/logo files into this folder.
